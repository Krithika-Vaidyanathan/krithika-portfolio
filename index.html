Expense Tracker

Angular

A full-stack expense tracking application that allows users to create budgets, record expenses, and monitor spending through real-time progress indicators and charts.
The frontend is built using Angular with a service-driven reactive architecture using RxJS, while Supabase provides authentication and a PostgreSQL database for secure data storage. The application supports session persistence, multi-user data isolation, and real-time UI updates across components.
Additional features include exporting financial reports in CSV, Excel, and PDF formats and a feedback system for collecting user suggestions. The frontend is deployed on Netlify for fast global delivery and seamless CI/CD from GitHub.
